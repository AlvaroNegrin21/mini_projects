# 🐍 Mini Projects — Python

A collection of small Python projects built while learning the language. Each folder is an independent project that practices a different concept.

## 📂 Projects

| Project | Description | Concepts practiced |
|---------|-------------|---------------------|
| `grades_calculator` | Calculates the average, max, and min of several grades | Lists, loops, validation |
| `shopping_cart` | Shopping cart with automatic discount | Dictionaries, loops |
| `task_manager` | Console-based task manager | Functions, control flow |
| `task_manager_v2` | Task manager with JSON persistence | Files, JSON, modules |
| `temperature_converter` | Converts between Celsius, Fahrenheit, and Kelvin | Functions, modules |
| `text_analyzer` | Analyzes text: word count, vowels, most repeated word | String manipulation |
| `bank` | Simple bank account simulator | Classes & OOP |
| `contact_manager` | Address book with search and persistence | OOP + JSON combined |
| `guessing_game` | Number guessing game | Random module, control flow |
| `weather_app` | Checks the current weather for any city | External APIs, requests |

## 🛠️ How to run

Each project has its own main file. For example:

```bash
cd grades_calculator
python grades.py
```

Some projects (`weather_app`) require external dependencies. Install them with:

```bash
pip install requests
```

## 📚 About this repository

Projects created as progressive practice in Python, ranging from basic concepts to object-oriented programming, file handling, and API consumption.
