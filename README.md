# 🐍 Mini Projects — Python

Colección de pequeños proyectos hechos mientras aprendía Python. Cada carpeta es un proyecto independiente que practica un concepto distinto.

## 🛠️ Cómo ejecutar

Cada proyecto tiene su propio archivo principal. Por ejemplo:

```bash
cd grades_calculator
python grades.py
```

## 📚 Sobre este repositorio

Proyectos creados como práctica progresiva de Python, desde conceptos básicos hasta programación orientada a objetos, manejo de archivos y consumo de APIs.
